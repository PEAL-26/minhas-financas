# Minhas Finanças | PEALSystems

## [Minhas Finanças]

O **Minhas Finanças** é um sistema completo para gerenciar suas finanças pessoais de maneira eficaz. Desenvolvido pela PEALSystems, este projeto visa ajudar você a alcançar controle financeiro, planejamento e tranquilidade em sua vida financeira.

## Funcionalidades

### Planos Financeiros

- Criar planos financeiros mensais

### Despesas

- Controle das despesas (únicas, diárias e mensais)

### Rendas

- Controle das rendas (únicas, diárias e mensais)

### Necessidades

- Gerenciar as necessidades e planear a melhor forma para satisfaze-las

## Como Começar

1. Clone este repositório para o seu ambiente local:

   ```
   git https://github.com/PEAL-26/minhas-financas.git
   ```

2. Configure o ambiente de desenvolvimento de acordo com as instruções do arquivo [INSTALL.md](INSTALL.md).

3. Execute o aplicativo e comece a usar as funcionalidades para melhorar seu controle financeiro pessoal.
